# AIOS Enterprise

## Purpose

AIOS Enterprise is the AI Operating System for this repository.

It is the single source of truth for:

- Architecture
- Engineering standards
- Trading standards
- Persistent memory
- AI agents
- Governance
- Documentation
- Workflows

Every AI model (Claude, GPT, Qwen, Gemini, Codex, Cursor, Roo, Cline, OpenHands) must read AIOS before making changes.

## Rules

1. Never guess.
2. Read project memory first.
3. Verify before changing code.
4. Protect trading capital above all else.
5. Every feature requires:
   - Tests
   - Documentation
   - Security review
   - Risk review
6. Never hardcode secrets.
7. Never delete code without explicit approval.
8. Keep business logic separate from UI.
9. Update documentation after every completed feature.
10. Leave the repository in a better state than you found it.
